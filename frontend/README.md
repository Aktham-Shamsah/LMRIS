# LRMIS Frontend

React + Vite frontend for the LRMIS project.

## Run

```powershell
cd ...\lrmis\frontend
npm install
npm run dev
```

The app expects the backend at `http://localhost:8000`. Override with:

```powershell
$env:VITE_API_BASE_URL="http://localhost:8000"
npm run dev
```

## Screens

- User selection
- Applicant dashboard
- Submit application
- Track application
- Staff dashboard
- Application management table
- Application details
- Surveyor task list
- Survey task execution
- Live parcel map
- Analytics dashboard
- Certificate view
